## Langflow AI Agents

This repository includes custom tools and AI agents built using Langflow to streamline the Quality Assurance process. The flows can be found in the `Langflow` folder and are ready to be imported into any Langflow environment.

- **QA Buddy Agent:** An interactive AI assistant configured to answer domain-specific queries related to Quality Assurance.
- **Bug Classifier Agent:** An automated triage tool that analyzes bug reports, classifies them by severity and technical category, and checks for missing reproduction information.

For more details on how these flows work, please see the [Langflow README](./Langflow/README.md).
